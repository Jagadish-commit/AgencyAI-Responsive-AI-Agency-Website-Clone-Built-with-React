# AgencyAI – Responsive AI Agency Website

A modern and fully responsive AI agency website clone built with React and Vite. The project showcases clean UI design, reusable components, and mobile-first responsive layouts.

## Features

- Fully responsive design
- Built with React
- Modern UI/UX layout
- Reusable components
- Mobile-friendly navigation
- Fast and optimized performance
- Clean and organized code structure

## Tech Stack

- React.js
- JavaScript
- Tailwind CSS
- Vite


## Purpose

This project was created for learning, practice, and portfolio purposes to improve frontend development and React skills.
